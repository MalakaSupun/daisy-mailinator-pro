# Email Body Structure

## Success Email

1. Greeting and execution context
2. Summary metrics table
3. Key highlights
4. Output links
5. Exception summary, when applicable
6. Run ID
7. Standard RPA footer

## No Data Email

1. Greeting
2. Successful completion statement
3. No invoices available statement
4. Output or previous output links, when available
5. Run ID
6. Standard RPA footer

## Failure Email

1. Greeting and failure statement
2. Failure detail table
3. Impact counts
4. Required action
5. Log/output links
6. Run ID
7. Standard RPA footer

## Exception Alert Email

1. Greeting and exception statement
2. Exception summary table
3. Exception report/log links
4. Run ID
5. Standard RPA footer
